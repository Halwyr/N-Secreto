# 🎮 Jogo do Número Secreto 🎮

## 📝 **Sobre:**
Projeto desenvolvido durante os cursos de **Lógica de Programação** da **Alura**. O jogo consiste em adivinhar um número secreto definido aleatoriamente pelo programa.
